# HOTPANTS Options

Complete list of HOTPANTS command-line options (`hotpants -h`, version 5.1.11),
for reference when tuning the [`hotpants_extra`](task_config.md) field. STDWeb
sets the input/output, mask, noise, gain and kernel-sizing options automatically;
the remaining options can be overridden through `hotpants_extra` (using the
option name without its leading dash).
